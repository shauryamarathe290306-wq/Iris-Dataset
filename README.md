# Iris Dataset Visualization

## Project Overview

This project is part of the **Synent Technologies Data Science Internship Program - Basic Task 2: Data Visualization**.

The objective of this project is to visualize the Iris dataset using different chart types and identify patterns among different flower species.

---

## Problem Statement

Analyze and visualize the Iris dataset by creating:

* Bar Chart
* Histogram
* Scatter Plot
* Feature Comparison

The goal is to understand relationships between features and identify patterns among flower species.

---

## Dataset Information

**Dataset Name:** Iris Dataset

The dataset contains measurements of iris flowers from three species:

* Setosa
* Versicolor
* Virginica

### Features

| Feature      | Description          |
| ------------ | -------------------- |
| Sepal Length | Length of sepal (cm) |
| Sepal Width  | Width of sepal (cm)  |
| Petal Length | Length of petal (cm) |
| Petal Width  | Width of petal (cm)  |
| Target       | Species Label        |

Total Records: **150**

---

## Tools and Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

---

## Visualizations Created

### 1. Bar Chart

Shows the number of flowers in each species.

### 2. Histogram

Displays the distribution of sepal length values.

### 3. Scatter Plot

Compares sepal length and petal length across species.

### 4. Feature Comparison

Compares average measurements of different flower species.

---

## Key Insights

* The dataset contains three flower species.
* Each species has 50 observations, making the dataset balanced.
* Setosa has the smallest petal dimensions.
* Virginica has the largest petal dimensions.
* Petal length and petal width are the most useful features for distinguishing species.
* Scatter plots clearly show separation between Setosa and the other species.
